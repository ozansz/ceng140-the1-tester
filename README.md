# ceng140-the1-tester
CENG140 THE1 Ucoluk Edition Unofficial Tester

## Usage
```shell
git clone https://github.com/ozansz/ceng140-the1-tester
cd ceng140-the1-tester

python3 tester.py --json sample_test.json the1executable
```
